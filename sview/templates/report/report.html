Sentinel Report

Some data that we show on Sentinel View pages are constrained to narrow results fixed on short time perspective.

In order to bring data to more broad perspective and context we release Sentinel Report in PDF form on monthly basis.

Sentinel Report repository

How the reports are composed

For the tables like Password deltas and Port trends comparing old and new results we start with data gathered on a topic for passed month and compare them with same samples in the month before. For other type of data, we run queries to find outmean or median for the month.

We gather our data from internal and also public sources.
